# Emacs.d
My Emacs configuration

# Installation
Clone my repo to your .emacs.d directory. For example, on Linux do this:
```
git clone https://github.com/liferooter/emacs.d ~/.emacs.d
```

Then open Emacs and do: `M-x all-the-icons-install-fonts`
